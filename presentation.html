04 / IDENTITY RESOLUTION

The matcher: deterministic, then fuzzy.

Pass 1: exact match on normalized email, E.164 phone, and cross-source FKs. Pass 2: rapidfuzz over (name, street, city) blocked by (last_initial, region, birth_year). Survivorship is field-level: every value on the master row carries a source and a confidence. Record confidence = floor of populated-field confidences.

≥ 0.90
Auto-merge
0.70 — 0.89
Stewardship queue
< 0.70
Keep separate

Scoreboard mock: top-left counter "5 awaiting review", side-by-side diff of David Lee vs D Lee +amazon, two big buttons — Merge / Keep Separate — field-level confidence tooltips.

05 / QUALITY GATE

The hook that blocks the curated zone.

Schema drift, null explosion, volume anomaly, referential integrity. Each check is Python code, not a prompt instruction. A PreToolUse hook intercepts every Write into warehouse/curated/.

 Claude Code wants to Write → warehouse/curated/customer_master.parquet
                                       |
                                       v
                .claude/hooks/curated_gate.sh
                                       |
                  +--------------------+---------------------+
                  |                                          |
              contracts pass                         contracts fail
                  |                                          |
              exit 0                              exit 2 + stderr to Claude
                  |                                          |
            write happens                            write blocked

Hooks for deterministic guardrails. Prompts for probabilistic preferences. The split is explicit and lives in the ADR.

07 / EVAL

A single defensible number.

20+ golden pairs in backend/tests/golden_pairs.csv, stratified into easy, hard, boundary, and negative. Runs in CI on every PR.

MetricWhat it answers
PrecisionOf merges we auto-approved, how many were right?
RecallOf real duplicates, how many did we catch?
False-confidence-rateHow often did we say "≥ 0.90" and be wrong?

Headline cases: Sean O'Brien (4 sources), Maria Hernandez-Garcia (4), Jürgen Müller mojibake duplicate, David/Dave/D Lee + amazon boundary, Sean Williams ≠ Sean Rodriguez, Jurgen Schmidt Berlin ≠ Köln — the matcher must not over-merge.
